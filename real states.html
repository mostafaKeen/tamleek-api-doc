Real Estate API Documentation

GET /api/real-estates

Get a paginated list of real estate items. Role-based filtering applies.

Query Parameters

ParameterTypeDescription
pageintegerPage number (default: 1)
limitintegerItems per page (default: 10)
searchstringSearch by title

Response (200 OK)

{
    "data": [ { ... } ],
    "page": 1,
    "limit": 10,
    "total": 50,
    "last_page": 5
}

GET /api/real-estates/{userId}

Get a user's portfolio of real estates. Overrides each item's owner with the specified user.

Query Parameters

ParameterTypeDescription
pageintegerPage number (default: 1)
limitintegerItems per page (default: 10)
searchstringSearch by title

Response (200 OK)

{
    "data": [ { ... } ],
    "page": 1,
    "limit": 10,
    "total": 20,
    "last_page": 2
}

POST /api/real-estates

Create a new real estate item. Authenticated user becomes owner.

Request Body

{
    "title": "Villa in Dubai Marina",
    "price": 1500000,
    "currency": "USD",
    "stage": "NEW",
    "source": "CALL",
    "start_date": "2026-01-06T03:00:00+03:00",
    "end_date": "2026-01-15T03:00:00+03:00",
    "property_details": "Luxury villa with sea view",
    "property_files": null
}

Response (200 OK)

{
    "result": { "id": 101, "fields": { ... } }
}

PUT /api/real-estates/{id}

Update a real estate item. Only accessible if user has permission.

Request Body

{
    "title": "Updated Villa Title",
    "price": 1550000,
    "stage": "IN_PROGRESS",
    "property_details": "Updated details"
}

Response (200 OK)

{
    "result": { "id": 101, "fields": { ... } }
}

Response (403 Forbidden)

{
    "message": "Forbidden"
}

DELETE /api/real-estates/{id}

Delete a real estate item. Only accessible if user has permission.

Response (200 OK)

{
    "success": true
}

Response (403 Forbidden)

{
    "message": "Forbidden"
}
