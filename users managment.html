User Management API Documentation

GET /api/users

Description: List users with search and pagination.

Query Parameters

ParameterTypeDescription
pageintegerPage number (default 1)
per_pageintegerItems per page (default 10)
searchstringSearch by name, email, Instagram, TikTok, or WhatsApp links

Response

{
    "data": [
        {
            "id": 1,
            "name": "John Doe",
            "email": "john@example.com",
            "role": "agent",
            "phone": "+123456789",
            "agent_id": null,
            "team_lead_id": 2,
            "instagram_link": "https://instagram.com/john",
            "tiktok_link": "https://tiktok.com/@john",
            "whatsapp_link": "https://wa.me/123456789"
        }
    ],
    "total": 1,
    "page": 1,
    "per_page": 10
}

POST /api/users

Description: Create a new user (permissions depend on role).

Request Body

{
    "name": "Jane Doe",
    "email": "jane@example.com",
    "password": "password123",
    "role": "agent",
    "phone": "+123456789",
    "instagram_link": "https://instagram.com/jane",
    "tiktok_link": "https://tiktok.com/@jane",
    "whatsapp_link": "https://wa.me/123456789"
}

{
    "id": 5,
    "name": "Jane Doe",
    "email": "jane@example.com",
    "role": "agent",
    "phone": "+123456789",
    "agent_id": null,
    "team_lead_id": 2,
